# Задача 5 из первого блока задач
Написать рекурсивную функцию для расчета степени n вещественного числа a (n — натуральное число).

**Рекурсия** – это вызов функцией самой себя. 
```python
def power(base, exp):
    if (exp == 1):
        return (base)
    if (exp != 1):
        return (base * power(base, exp - 1))
base = int(input("Введите число: "))
exp = int(input("Введите степень: "))
print("Результат равен:", power(base, exp))
```   
**Ход реализации**
> Принимаем на вход число и значение его степени, записываем их в отдельные переменные ``` base ``` и  ``` exp ``` соответственно. Передаем эти переменные в качестве аргументов в рекурсивную функцию ``` power() ```. Определяем в качестве базового условия рекурсии ``` exp == 1 ```. В этом случае функция выводит само число из переменной  ``` base  ```, так как любое число в степени 1 равно само себе. В противном случае мы выводим произведение числа на эту же функцию, в которой второй аргумент (степень) уменьшен на единицу: ```python base * power(base, exp - 1) ```. Таким образом мы накапливаем произведение числа, хранящегося в переменной base, самого на себя ровно exp раз, что эквивалентно возведению числа base в степень exp. Выводим конечный результат на экран.

```python
Введите степень: 5

```
